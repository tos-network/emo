# HTTP/RPC Clients

Emo provides official HTTP RPC  (`/api/v0`) clients for selected languages:

| Language |     Package Name    | Github Repository                          |
|:--------:|:-------------------:|--------------------------------------------|
| JS       | emo-rpc-client     | https://github.com/ipfs/js-emo-rpc-client |
| Go       | `rpc`               | [`../client/rpc`](../client/rpc)           |

There are community-maintained libraries for other languages,
but the Emo team does provide support for them, YMMV:

- https://docs.ipfs.tech/reference/emo-rpc-cli/
